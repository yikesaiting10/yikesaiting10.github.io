### 字符串逆序
时间复杂度O(n)，空间复杂度O(1)  
string可以直接使用reverse(string.begin(), string.end())
```c++
void reverse(char *p)
{
    char *l = p;
    char *r = p;
    while(r)
    {
        r++;
    }
    r--;
    while(r>l)
    {
        char t = *l;
        *l++ = *r;
        *r-- = t;
    }
}
```
