### 二叉树的前序遍历  
给定一个二叉树，返回它的前序遍历。
示例:
```
输入: [1,null,2,3]  
   1
    \
     2
    /
   3 
输出: [1,2,3]
```
进阶: 递归算法很简单，你可以通过迭代算法完成吗？  
**思路：**用栈模拟递归的过程。  
```c++
vector<int> preorderTraversal(TreeNode* root) {
        vector<int> res;
        if(!root) return res;
        stack<TreeNode*> s;
        s.push(root);
        while(!s.empty())
        {
            TreeNode* temp = s.top();
            res.push_back(temp->val);
            s.pop();
            if(temp->right) s.push(temp->right);
            if(temp->left) s.push(temp->left);
        }
        return res;
    }
```
