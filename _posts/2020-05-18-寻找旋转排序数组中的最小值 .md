### 寻找旋转排序数组中的最小值  
假设按照升序排序的数组在预先未知的某个点上进行了旋转。  
( 例如，数组 [0,1,2,4,5,6,7] 可能变为 [4,5,6,7,0,1,2] )。  
请找出其中最小的元素。  
你可以假设数组中不存在重复元素。  
示例 1:  
```
输入: [3,4,5,1,2]
输出: 1
```
示例 2:  
```
输入: [4,5,6,7,0,1,2]
输出: 0
```
**思路：**二分法：nums[mid] > nums[r]时说明最小值在[mid+1, r]之间，否则在[l, mid]之间。时间复杂度O(n)。  
```c++
int findMin(vector<int>& nums) {
        int l = 0;
        int r = nums.size() - 1;
        while(l < r)
        {
            int mid = l + (r - l) / 2;
            if(nums[mid] > nums[r])
                l = mid + 1;
            else
                r = mid;
        }
        return nums[l];
    }
```
