### 二进制求和  
给你两个二进制字符串，返回它们的和（用二进制表示）。  
输入为 非空 字符串且只包含数字 1 和 0。  
示例 1:  
```
输入: a = "11", b = "1"
输出: "100"
```
示例 2:  
```
输入: a = "1010", b = "1011"
输出: "10101"
```
提示：  
每个字符串仅由字符 '0' 或 '1' 组成。  
1 <= a.length, b.length <= 10^4  
字符串如果不是 "0" ，就都不含前导零。  
**思路：**逐位相加。[位操作]()无进位和为a异或b:a^b，进位为a与b左移一位:(a&b)<<1,进位和为a^b+(a&b)<<1重新进行两个二进制求和，直到进位为0。  
```c++
//逐位相加
string addBinary(string a, string b) {
        int n = a.size();
        int m = b.size();
        string res = "";
        int flag = 0;
        while(n > 0 && m > 0)
        {
            n--;
            m--;
            if(a[n] == '0' && b[m] == '0')
            {
                if(flag == 0)
                {
                    res += '0';
                    flag = 0;
                }
                else
                {
                    res += '1';
                    flag = 0;
                }
            }
            if((a[n] == '0' && b[m] != '0') || (a[n] != '0' && b[m] == '0'))
            {
                if(flag == 0)
                {
                    res += '1';
                    flag = 0;
                }
                else
                {
                    res += '0';
                    flag = 1;
                }
            }
            if(a[n] != '0' && b[m] != '0')
            {
                if(flag == 0)
                {
                    res += '0';
                    flag = 1;
                }
                else
                {
                    res += '1';
                    flag = 1;
                }
            }
        }
        while(n > 0)
        {
            n--;
            if(a[n] == '0')
            {
                if(flag == 0)
                {
                    res += '0';
                    flag = 0;
                }
                else
                {
                    res += '1';
                    flag = 0;
                }
            }
            if(a[n] != '0')
            {
                if(flag == 0)
                {
                    res += '1';
                    flag = 0;
                }
                else
                {
                    res += '0';
                    flag = 1;
                }
            }
        }
        while(m > 0)
        {
            m--;
            if(b[m] == '0')
            {
                if(flag == 0)
                {
                    res += '0';
                    flag = 0;
                }
                else
                {
                    res += '1';
                    flag = 0;
                }
            }
            if(b[m] != '0')
            {
                if(flag == 0)
                {
                    res += '1';
                    flag = 0;
                }
                else
                {
                    res += '0';
                    flag = 1;
                }
            }
        }
        if(flag == 1)
            res += '1';
        reverse(res.begin(), res.end());
        return res;
    }
```
