### 字符串逆序  

O(n)时间复杂度，O(1)时间复杂度  
string类型可直接使用reverse(string.begin(), string.end())  
```c++
void reverse(char *p)
{
    char *l = p;
    char *r = p;
    while(r)
    {
        r++;
    }
    r--;
    while(r>l)
    {
        char t = *l;
        *l++ = *r;
        *r-- = t;
    }
}
```
